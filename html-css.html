Web Technologies - Unit I

HTML and CSS: Structure, Styling, and Design of Web Pages

1. Introduction to HTML (HyperText Markup Language)

HTML is the standard language for creating web pages and applications. It structures content using tags and elements interpreted by browsers.

  • HTML stands for HyperText Markup Language.
  • Used to create web pages and web applications.
  • Not a programming language, but a markup language.
  • Forms the backbone of every website on the internet.

HyperText

“Text within text.” HyperText allows linking between web pages using hyperlinks (<a href>).

Markup Language

A language used to structure and format documents using tags and elements.

Basic Structure of an HTML Page

HTML Page Structure – HTML, Head, Body, and Elements

<!DOCTYPE html> <html> <head> <title>Page Title</title> </head> <body> <!-- Content goes here --> </body> </html>

2. Common HTML Tags

HTML uses tags to define page elements. Tags are enclosed in angle brackets < > and often come in pairs.

  • <h1> to <h6> — Headings
  • <p> — Paragraph
  • <a href="url">Link</a> — Anchor (link)
  • <img src="url" alt="text"> — Image
  • <ul> / <ol> — Lists
  • <table> — Table
  • <form> — Form for input
<h1>Welcome to HTML</h1> <p>This is a paragraph.</p> <a href="https://example.com">Click here</a> <img src="image.jpg" alt="Sample">

3. HTML Lists

Ordered and Unordered Lists

<ul> <li>HTML</li> <li>CSS</li> </ul> <ol> <li>Item 1</li> <li>Item 2</li> </ol>

4. Forms in HTML

Forms collect user input such as text, passwords, selections, and files.

<form> User Name: <input type="text"><br> Password: <input type="password"><br> Gender: <input type="radio" name="g">Male <input type="radio" name="g">Female<br> Hobbies: <input type="checkbox">Cricket <input type="checkbox">Chess<br> <input type="submit" value="Submit"> </form>

5. HTML Tables

Used to display data in rows and columns.

<table border="1"> <tr><th>S.No</th><th>Name</th><th>Roll</th></tr> <tr><td>1</td><td>Swapna</td><td>101</td></tr> </table>

6. Introduction to CSS (Cascading Style Sheets)

CSS controls the presentation and design of HTML pages — colors, layout, fonts, and responsiveness.

Syntax

selector { property: value; }

Advantages of CSS

  • Faster page load
  • Easy formatting and maintenance
  • Responsive design
  • Reusability across multiple pages

7. Types of Style Sheets

  • Inline CSS: inside an element using style attribute
  • Internal CSS: within <style> in <head>
  • External CSS: linked via <link> tag
<h1 style="color:blue;">Inline CSS Example</h1>
<style> h1 {color:green;} p {font-size:18px;} </style>
<link rel="stylesheet" href="style.css">

8. CSS Selectors

  • Element Selector: h1 { color: red; }
  • Class Selector: .title { color: blue; }
  • ID Selector: #main { text-align: center; }
  • Group Selector: h1, p { color: purple; }

9. CSS Properties

Text and Background

p { color: red; font-size: 18px; background-color: lightyellow; }

Border and Margin

div { border: 2px solid blue; margin: 20px; }

Table Styling

table, th, td { border: 1px solid black; border-collapse: collapse; padding: 10px; }

10. HTML Media

HTML supports embedding of images, audio, and video.

<video width="320" height="240" controls> <source src="movie.mp4" type="video/mp4"> </video> <audio controls> <source src="audio.mp3" type="audio/mpeg"> </audio>

11. HTML Graphics (Canvas & SVG)

Used for rendering graphics, shapes, and animations in HTML5.

Canvas

<canvas id="myCanvas" width="200" height="100"></canvas> <script> var c = document.getElementById("myCanvas"); var ctx = c.getContext("2d"); ctx.fillStyle = "blue"; ctx.fillRect(20, 20, 150, 50); </script>

SVG (Scalable Vector Graphics)

<svg width="100" height="100"> <circle cx="50" cy="50" r="40" stroke="green" stroke-width="4" fill="yellow" /> </svg>
