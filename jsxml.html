Web Technologies - Unit II

JavaScript and XML: Dynamic Web Pages and Data Representation

1. Introduction to JavaScript

JavaScript is a client-side scripting language used to make web pages dynamic and interactive. It runs inside web browsers and can manipulate HTML, CSS, and handle user interactions.

  • JavaScript is an interpreted language.
  • Developed by Brendan Eich in 1995.
  • Follows Object-Based and Event-Driven paradigms.
  • Can be embedded directly in HTML.

JavaScript Flow: HTML → Browser → JavaScript Engine

<script> document.write("Welcome to JavaScript!"); </script>

2. JavaScript Syntax and Variables

JavaScript statements are separated by semicolons and can declare variables using var, let, or const.

let name = "John"; var age = 22; const country = "India";
  • var – function scoped
  • let – block scoped
  • const – cannot be reassigned

3. Data Types and Operators

JavaScript supports primitive and object data types.

  • Primitive: String, Number, Boolean, Null, Undefined
  • Objects: Array, Function, Date, etc.

Operators:

+, -, *, /, %, ++, --, ==, ===, &&, ||, !

let a = 5, b = 2; console.log(a + b); // 7 console.log(a * b); // 10 console.log(a == b); // false

4. Control Statements

Used to control program flow using decisions and loops.

if (age >= 18) { console.log("Adult"); } else { console.log("Minor"); } for (let i=1; i<=5; i++) { console.log(i); }

5. Functions in JavaScript

Functions are reusable blocks of code that perform a specific task.

function greet(name) { return "Hello, " + name + "!"; } console.log(greet("Alice"));

Arrow Function Example

const add = (x, y) => x + y; console.log(add(5, 10));

6. Arrays and Objects

Arrays

let fruits = ["Apple", "Banana", "Mango"]; console.log(fruits[1]); // Banana

Objects

let student = { name: "Rahul", roll: 101, course: "CSE" }; console.log(student.name);

7. String and Math Functions

let str = "Web Technologies"; console.log(str.length); console.log(str.toUpperCase()); let num = Math.sqrt(25); console.log(num); // 5

8. Events in JavaScript

Events are user interactions that trigger scripts (click, mouseover, keypress, etc.).

JavaScript Event Handling Flow

<button onclick="alert('Button Clicked!')">Click Me</button>

9. Error Handling

JavaScript handles runtime errors using try–catch–finally blocks.

try { let result = 10 / 0; throw "Custom Error Example"; } catch (error) { console.log("Error: " + error); } finally { console.log("Execution Completed"); }

10. Introduction to XML (eXtensible Markup Language)

XML is a markup language used to store and transport data. It is both human-readable and machine-readable.

  • Self-descriptive and platform-independent
  • Used for data interchange (like JSON)
  • Defines user-specific tags

XML Hierarchical Tree Structure

<note> <to>Alice</to> <from>Bob</from> <message>Hello, Welcome to XML!</message> </note>

11. XML Syntax Rules

  • Must have a root element.
  • Tags are case-sensitive.
  • All tags must be properly nested.
  • Attribute values must be quoted.

12. DTD (Document Type Definition)

Defines the structure and rules for XML documents.

<!DOCTYPE note [ <!ELEMENT note (to,from,message)> <!ELEMENT to (#PCDATA)> <!ELEMENT from (#PCDATA)> <!ELEMENT message (#PCDATA)> ]>

13. XML Schema

XML Schema (XSD) defines the structure of an XML document using data types and constraints.

<xs:element name="student"> <xs:complexType> <xs:sequence> <xs:element name="name" type="xs:string"/> <xs:element name="roll" type="xs:integer"/> </xs:sequence> </xs:complexType> </xs:element>

14. XML Parsers

XML Parsers are used to read and manipulate XML documents.

  • DOM Parser: Loads entire XML into memory as a tree.
  • SAX Parser: Reads XML sequentially (event-based).
