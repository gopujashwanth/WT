Dynamic web content generation using JSP technology — the simplified way to build server-side applications.

Introduction to JSP

JSP (Java Server Pages) allows embedding Java code into HTML pages to create dynamic web applications. It is an extension of Servlets, offering easier development and built-in features like Expression Language and JSTL.

Advantages of JSP over Servlets

  • Faster development — no recompilation required.
  • Less code and easy to maintain.
  • Supports Expression Language and Tag Libraries.
  • Automatically converted into Servlets by the container.

JSP Life Cycle

The JSP lifecycle defines how a JSP page is created, executed, and destroyed by the web container.

  1. Translation of JSP Page
  2. Compilation of JSP Page
  3. Classloading
  4. Instantiation
  5. Initialization (jspInit())
  6. Request Processing (_jspService())
  7. Destroy (jspDestroy())

JSP Life Cycle: From translation to destruction

public void jspInit() { ... } // Called once during initialization public void _jspService() { ... } // Invoked for each client request public void jspDestroy() { ... } // Called before JSP is unloaded

JSP Elements

Scripting Elements

  • Scriptlet: <% Java code %>
  • Expression: <%= value %>
  • Declaration: <%! variable/method %>
<html> <body> <% out.print("Hello JSP!"); %> </body> </html>

Directive Elements

Directives provide instructions to the container about page settings and behavior.

  • Page Directive: <%@ page import="java.util.Date" %>
  • Include Directive: <%@ include file="header.html" %>
  • Taglib Directive: <%@ taglib uri="..." prefix="..." %>

Action Elements

Action tags provide control over page flow and JavaBean interaction.

  • <jsp:forward> — forwards request to another page.
  • <jsp:include> — includes another resource dynamically.
  • <jsp:useBean> — locates or instantiates a bean.
<jsp:useBean id="user" class="com.demo.User" />
<jsp:setProperty name="user" property="username" value="Admin" />
<jsp:getProperty name="user" property="username" />

Database Access using JSP

<%@ page import="java.sql.*" %>
<%
try {
Class.forName("com.mysql.jdbc.Driver");
Connection con = DriverManager.getConnection("jdbc:mysql://localhost:3306/db","root","");
Statement st = con.createStatement();
ResultSet rs = st.executeQuery("SELECT * FROM student");
while(rs.next()) {
out.println(rs.getString(1));
}
con.close();
} catch(Exception e){ out.println(e); }
%>
